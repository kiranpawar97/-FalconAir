# ✈ Airline Reservation Management System

A web-based Airline Reservation Management System built using PHP, MySQL, Bootstrap and XAMPP.

## 🚀 Live Demo
[View Live Project](https://yourairline.gamer.gd)

## 📌 Features
- User Registration
- User Login Authentication
- Search Flights
- Add Flights
- Ticket Booking
- View My Booked Tickets
- Cancel Ticket
- Admin Flight Management
- Responsive Bootstrap UI

## 🛠 Tech Stack
- PHP
- MySQL
- XAMPP
- Bootstrap
- HTML/CSS
- phpMyAdmin

## 📂 Project Modules
### User Module
- Register/Login
- Search flights
- Book tickets
- Cancel bookings

### Admin Module
- Add flights
- Manage bookings
- View reservations

## 🗄 Database Tables
- users
- flights
- bookings

## ⚙ Installation
1. Clone repository

```bash
git clone https://github.com/yourusername/airline-reservation-system.git
```

2. Move project into:

```bash
xampp/htdocs/
```

3. Start Apache and MySQL in XAMPP

4. Import database:

```bash
airline_db.sql
```

5. Update database credentials in:

```bash
db.php
```

6. Run:

```bash
http://localhost/airline_reservation
```
